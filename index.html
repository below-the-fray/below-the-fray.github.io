Introduction

LaTeX documents have a very particular rhythm: generous margins, first‑line paragraph indentation, numbered sections, and figure/table captions. This single‑file HTML template mimics that look‑and‑feel for the web.

Mathematics can be rendered with KaTeX or MathJax. For example, the quadratic formula can be typeset inline like \(x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\) or as a centered equation below.

\[e^{i\pi} + 1 = 0\]

Background

Section counters and subsection counters are implemented with CSS counters, so headings are automatically numbered. Use <h2> for sections and <h3> for subsections.

Related Work

References can be presented in a numbered list at the end of the document, similar to a LaTeX thebibliography environment1.
